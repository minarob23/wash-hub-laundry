# 🧺 نظام إدارة المغسلة - البدء السريع

## ⚡ تشغيل البرنامج بسرعة

### الطريقة 1: استخدام Launcher (موصى به)
1. انقر نقراً مزدوجاً على `launcher.bat`
2. اختر الخيار المناسب من القائمة

### الطريقة 2: التشغيل المباشر
1. انقر نقراً مزدوجاً على `run.bat`

### الطريقة 3: يدوياً
```bash
python main.py
```

---

## 📋 قبل البدء

### 1. تثبيت Python
- قم بتحميل Python 3.8 أو أحدث من [python.org](https://www.python.org/downloads/)
- أثناء التثبيت، تأكد من تفعيل:
  - ☑️ **Add Python to PATH**

### 2. تثبيت المكتبات
قم بأحد الخيارات التالية:

**الخيار أ - استخدام launcher:**
- شغّل `launcher.bat`
- اختر [2] تثبيت المكتبات المطلوبة

**الخيار ب - استخدام install.bat:**
- انقر نقراً مزدوجاً على `install.bat`

**الخيار ج - يدوياً:**
```bash
pip install -r requirements.txt
```

## 📁 ملفات المشروع

### ملفات التشغيل:
- `launcher.bat` - قائمة تفاعلية (موصى به) ⭐
- `run.bat` - تشغيل مباشر
- `install.bat` - تثبيت المكتبات

### ملفات البرنامج:
- `main.py` - الملف الرئيسي
- `database.py` - إدارة قاعدة البيانات
- `models.py` - نماذج البيانات
- `customer_manager.py` - إدارة العملاء
- `product_manager.py` - إدارة المنتجات

### ملفات التوثيق:
- `README.md` - ملف تعريفي
- `USER_GUIDE.md` - دليل المستخدم الكامل 📖
- `DEVELOPER_GUIDE.md` - دليل المطور
- `QUICKSTART.md` - هذا الملف

### ملفات أخرى:
- `requirements.txt` - قائمة المكتبات المطلوبة
- `laundry_system.db` - قاعدة البيانات (يتم إنشاؤها تلقائياً)

## 🎯 الاستخدام الأساسي

### 1️⃣ بدء فاتورة جديدة
```
1. اضغط "New Order" (أو ابدأ مباشرة)
2. أدخل رقم هاتف العميل
3. اختر نوع الخدمة (Wash & Press, Pressing Only, إلخ)
4. اضغط على المنتجات المطلوبة
5. اضغط "Delivery & Save"
```

### 2️⃣ البحث عن عميل
```
1. أدخل رقم الهاتف في حقل "Customer"
2. أو اضغط زر البحث (🔍)
3. اختر العميل من القائمة
```

### 3️⃣ استرجاع فاتورة
```
1. اضغط "Retrieve"
2. أدخل رقم الفاتورة
3. ستظهر جميع التفاصيل
```

## ❓ حل المشاكل الشائعة

### المشكلة: "Python غير مثبت"
**الحل:**
1. حمّل Python من [python.org](https://www.python.org/downloads/)
2. تأكد من تفعيل "Add Python to PATH" أثناء التثبيت

### المشكلة: "فشل تثبيت المكتبات"
**الحل:**
```bash
pip install --upgrade pip
pip install customtkinter Pillow
```

### المشكلة: "لا يمكن العثور على قاعدة البيانات"
**الحل:**
- قاعدة البيانات تُنشأ تلقائياً عند أول تشغيل
- تأكد من وجود صلاحيات الكتابة في المجلد

### المشكلة: "البرنامج يتوقف فجأة"
**الحل:**
1. شغّل البرنامج من Command Prompt لرؤية الأخطاء:
```bash
python main.py
```
2. تحقق من ملف `laundry_system.log` إن وجد

## 💾 النسخ الاحتياطي

### طريقة سهلة:
1. شغّل `launcher.bat`
2. اختر [5] نسخة احتياطية
3. سيتم حفظ الملف في مجلد `backups/`

### يدوياً:
```bash
copy laundry_system.db backups\laundry_system_backup.db
```

**يُنصح بعمل نسخة احتياطية يومياً!**

## 📊 أول مرة؟ جرب هذه الخطوات:

### التجربة الأولى:
1. شغّل البرنامج
2. اضغط "New Order"
3. جرب إضافة منتجات
4. اضغط "Clear" لمسح السلة
5. جرب نوع خدمة مختلف
6. لا تحفظ الفاتورة، فقط تجريب

### بعد التجربة:
1. من القائمة: **إدارة** > **إدارة العملاء**
2. أضف عميلين أو ثلاثة
3. من القائمة: **إدارة** > **إدارة المنتجات**
4. تحقق من المنتجات والأسعار
5. الآن أنت جاهز للاستخدام الحقيقي!
